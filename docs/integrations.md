# Integrations

Panduan mengintegrasikan ElysiaJS dengan berbagai framework, database, dan alat populer lainnya.

## 1. Meta-Frameworks

Elysia dapat dijalankan bersama dengan framework frontend modern:

- **Next.js**: Menjalankan Elysia di Route Handlers.
- **Astro**: Integrasi sebagai API Endpoint.
- **SvelteKit**: Sebagai server-side endpoint.
- **Nuxt**: Menggunakan Elysia di Server Route.
- [Dokumentasi Resmi - Nextjs](https://elysiajs.com/integrations/nextjs.html)
- [Dokumentasi Resmi - Astro](https://elysiajs.com/integrations/astro.html)

## 2. Database & ORM

Integrasi dengan sistem database yang populer:

- **Drizzle ORM**: Alat ORM TypeScript yang sangat cepat dan kompatibel.
- **Prisma**: Dukungan penuh untuk skema Prisma.
- [Dokumentasi Resmi - Drizzle](https://elysiajs.com/integrations/drizzle.html)
- [Dokumentasi Resmi - Prisma](https://elysiajs.com/integrations/prisma.html)

## 3. Runtimes

Dukungan runtime selain Bun:

- **Cloudflare Workers**: Deploy ke edge dengan infrastruktur Cloudflare.
- **Deno**: Dukungan untuk runtime Deno.
- **Node.js**: Cara menjalankan Elysia di lingkungan Node.js standar.
- [Dokumentasi Resmi - Cloudflare](https://elysiajs.com/integrations/cloudflare-worker.html)
- [Dokumentasi Resmi - Node.js](https://elysiajs.com/integrations/node.html)

## 4. Other Tools

- **AI SDK**: Integrasi dengan Vercel AI SDK.
- **React Email**: Menggunakan JSX untuk template email.
- **Better Auth**: Solusi autentikasi modern yang terintegrasi.
- [Dokumentasi Resmi - AI SDK](https://elysiajs.com/integrations/ai-sdk.html)

---

> [!TIP]
> Gunakan `Drizzle` jika Anda menginginkan performa maksimal dan keselarasan tipe data yang ketat dengan TypeScript.
