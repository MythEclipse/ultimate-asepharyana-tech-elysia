# Official Plugins

Katalog plugin resmi yang didukung langsung oleh ekosistem ElysiaJS.

## 1. CORS

Menambahkan dukungan _Cross-Origin Resource Sharing_ ke aplikasi Anda.

- [Dokumentasi Resmi - CORS](https://elysiajs.com/plugins/cors.html)

## 2. JWT

Plugin untuk manajemen _JSON Web Token_ dengan integrasi sinkron dam asinkron.

- [Dokumentasi Resmi - JWT](https://elysiajs.com/plugins/jwt.html)

## 3. Bearer

Mengekstrak token Bearer dari header Authorization.

- [Dokumentasi Resmi - Bearer](https://elysiajs.com/plugins/bearer.html)

## 4. Cron (Implemented)

Mengatur tugas terjadwal (_cron jobs_) di aplikasi Anda. Digunakan untuk heartbeat sistem setiap menit.

- [Dokumentasi Resmi - Cron](https://elysiajs.com/plugins/cron.html)

## 5. GraphQL

Mendukung Apollo Server dan Yoga GraphQL.

- [Dokumentasi Resmi - GraphQL](https://elysiajs.com/plugins/graphql-yoga.html)

## 6. HTML

Mengembalikan respons HTML dengan dukungan untuk JSX dan komponen server.

- [Dokumentasi Resmi - HTML](https://elysiajs.com/plugins/html.html)

## 7. Static

Melayani file statis dari direktori tertentu.

- [Dokumentasi Resmi - Static](https://elysiajs.com/plugins/static.html)

## 8. Server Timing (Implemented)

Menambahkan header `Server-Timing` untuk analisa performa jaringan. Aktif secara global untuk melacak latensi middleware dan handler.

- [Dokumentasi Resmi - Server Timing](https://elysiajs.com/plugins/server-timing.html)

---

> [!TIP]
> Selalu aktifkan `CORS` jika aplikasi Anda diakses dari domain frontend yang berbeda.
